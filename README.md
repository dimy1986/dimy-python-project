# Voucher Extraction Program Documentation

## Features
- Extracts various types of vouchers from multiple sources.
- Supports multiple voucher formats including PDF, images, and scanned documents.
- User-friendly interface and quick processing time.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dimy1986/dimy-python-project.git
   cd dimy-python-project
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Quick Start
To start the voucher extraction program, run the following command in your terminal:
```bash
python voucher_extraction.py
```

## Usage Examples
- To extract vouchers from a PDF file:
```bash
python voucher_extraction.py --file input.pdf
```
- To extract vouchers from an image file:
```bash
python voucher_extraction.py --file input.jpg
```

## Supported Voucher Types
- Coupons
- Gift Cards
- Discount Vouchers
- Loyalty Points

## Troubleshooting
- **Problem:** The program fails to read the input file.
  - **Solution:** Ensure the file path is correct and the file format is supported.
- **Problem:** The extracted data seems incorrect.
  - **Solution:** Check if the voucher image is clear and readable.
