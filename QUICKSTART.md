# 凭证提取工具 - 快速开始指南

## 📋 5分钟快速开始

### 步骤 1: 环境准备（2分钟）

```bash
pip install -r requirements.txt
```

### 步骤 2: 运行程序（1分钟）

```bash
python main.py --input D:\bills_folder --output results.xlsx
```

### 步骤 3: 打包成 EXE

```bash
build.bat
```

## 🎯 常见问题

- OCR识别效果差？确保输入图片清晰，DPI >= 200
- 如何修改输出格式？编辑 voucher_validator.py
- 支持批量处理吗？支持！指定文件夹即可
